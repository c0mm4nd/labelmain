# LabelMain

## Ready

latest golang

your mongodb altas

write `.env` (as [.env.example](./.env.example)) or set the `MONGO_URI` var

## Usage

1. bitcoinabuse

```
go install github.com/c0mm4nd/labelmain/bitcoinabuse@latest
./bitcoinabuse -init 
```

1. walletexplorer

```
go install github.com/c0mm4nd/labelmain/walletexplorer@latest
nano .env
./walletexplorer
```
